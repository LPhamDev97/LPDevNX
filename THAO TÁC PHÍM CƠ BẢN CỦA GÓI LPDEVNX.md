* MỞ HOMEBREW MENU :
- Di chuyển đến mục ALBUM (Nhấn A)

* MỞ MENU UBERHAND :
- Nhấn cùng lúc phím trừ và phím cộng trên joy-con.

* CHỈNH ÉP XUNG GAME :
- Mở chạy game cần ép xung > Game mở lên rồi thì mở menu UberHand > Chọn sys-clk-oc > Chọn Edit app profile > Chơi cắm dock thì chỉnh mục Docked, chơi cầm tay thì chọn mục Handheld (Tự chỉnh thông số 3 mục nhỏ bao gồm : CPU,GPU,RAM) > LẦN MỞ GAME CHƠI TIẾP TỤC KHÔNG CẦN THAO TÁC - CHỈ GAME MỚI CÀI CHƯA CHỈNH THÌ CẦN VÀO CHỈNH THÔI.

* XEM THÔNG SỐ FPS VÀ NHIỆT ĐỘ THÂN MÁY :
- Mở UberHand > Chọn Status Monitor > Chọn Micro > Vào game chơi sẽ hiện đầy đủ cả 2 thông số.

* ÉP CHẠY 60FPS :
- Mở chạy game cần ép xung > Game mở lên rồi thì mở menu UberHand > Chọn FPSLocker > Chọn dòng đầu tiên cho đến khi thấy số đạt 60FPS.

* BẬT CHEAT GAME :
(Yêu cầu cần chép cheat thủ công vào)
- Mở chạy game cần ép xung > Game mở lên rồi thì mở menu UberHand > Chọn EdiZon > Chọn Cheats.

* TẮT MÁY - RESTART MÁY KHÔNG CẦN NHẤN PHÍM NGUỒN :
- Mở UberHand > Nhấn mũi tên sang phải (Joy-con trái) > Dùng các mũi tên lên - xuống để chọn > Bên dưới cùng sẽ có 2 lựa chọn Reboot (Khởi động máy) hoặc Shutdown (Tắt máy).
